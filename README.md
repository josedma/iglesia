# ICCED Landing (React)

Landing page para iglesia basada en React + Vite.

## Requisitos

- Node.js 18+

## Ejecutar

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

## Estructura

- `src/components`: secciones reutilizables
- `src/data/content.js`: contenido centralizado
- `src/App.css`: sistema visual y estilos globales
